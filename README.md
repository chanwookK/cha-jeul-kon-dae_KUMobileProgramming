# 찾을건대
![찾을건ㄷ대](https://github.com/chanwookK/mobileProgramming3team/assets/92363927/74c3efa3-85e8-46eb-9692-60e61e622ef7)

## 프로젝트 소개

<p align="justify">
중앙 집중식 교내 분실물 찾기 서비스
<br>
<br>
  
### 프로젝트 동기/목표
‘일상속에서 불편함’ 을 해결하기 위해 어플리케이션 개발
그 불편함 중 하나는 교내에서 물건을 분실했을 때 무엇부터 해야 할지 어디에 문의해야 할지 등 여러가지 정보의 부재였습니다.

이 어플리케이션은 사용자들이 분실물을 찾는 데 필요한 정보를 제공하고, 자치기구가 발견한 분실물을 쉽게 등록할 수 있도록 기능을 제공하는 것이 목적입니다.   
</p>



<br>
